Play on snapshot 20w13b: https://www.youtube.com/watch?v=-gWUURjTaL0

Resourcepack:
	1. Open resourcepack folder: "Options" -> "Resource Packs" -> "Open Resource Pack Folder"
	2. Drag 20w14s_resourcepack.zip into the resourcepacks folder

Datapack:
	1. Open world's datapack folder: "Singleplayer" -> Click world -> "Edit" -> "Open World Folder" -> "datapacks"
	2. Drag 20w14s_datapack.zip into the datapacks folder

Don't install this on any worlds you care about!
